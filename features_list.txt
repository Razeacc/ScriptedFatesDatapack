## - DEV ROADMAP -
- v1.1 "Dinner's Up!" - Roleplaying Update, Chemistry Discipline. Custom Food/Drinks/Potions + Tier 2 Catalysts
- v1.2 "Tuning In" - Transmu-Tuning Discipline. Item Transfiguration and building assistance
- v1.3 "Magic on Metal" - Craftable Enchant update. Introduction of "Avalonian Steel", crafted using iron, echo shards, and books enchanted books. This material is the "Archa Engineering" Catalyst. When used alongside Transmu-Tuning, can essentially let you Recycle enchants into newer/better/specific enchants.
- v1.4 "Left Behind" - "History" Update. Changes/modifiers Sniffer Sniffing, Fishing, Archeology, Trading, and many more Loot Tables with new relics, lore tablets, etc. A new material is introduced too: "Innanimate Khueenite". When crafted with any meat, it becomes "Sentient Khueenite", which will be used in the next update.
- v1.5 "Right on Track" - Blank/Innate Memory Piece Extraction is now possible, alongside faster minecarts (maybe) and introduction of "Life-Bearing Discipline", letting you craft "golem assembly devices" using the new Sentient Khueenite introduced in the last update. Golems also drop Innate Khueenite now.

- v2.0 "FractureGraven" - New Sculk Raid/Purification mechanic. Players can either start a Sculk Raid to slowly purify an area of sculk while new sculken enemies attack them. Or, Players can instead *spread* Sculk, alerting all other players on the server to their location.
Spreading sculk this way will need an expensive resource to begin (maybe a nether star or a new drop from a boss in sculk purification raids?). It spreads sculk passively, but slowly, and can be easily broken by a single attack.
Meanwhile, purifying sculk needs Echo Sheets, a specific block to replace with (dirt, stone, or sand), and a Heart of the Sea (not a conduit). You basically surround the blocks and echo sheet with the Heart to make the Sculk Purifier. Placing it down on sculk begins the raid if enough sculk blocks are nearby. It purifies the area quickly (5-10 ish minutes) while waves of enemies try to break the conduit (it has 20 hearts like a player)
New resources obtained from these raids will allow for the creation of Memory Pieces using Blank/Innate Memorie Pieces. By using a new "Vessel Memory Interface", it'll open a new chest-like UI that lets you literally drag in and out Memory Pieces (it's tied to each player like an enderchest). Depending on what memory pieces are in here, will grant permament attribute boosts, like +1 HP, +1 Reach, +1 Speed, etc. (IDK if any of these are possible tho...)
- v2.1 "NetherMore" - Nether Update. New Nether structures, new history/lore similar to the "Left Behind" update, and maybe reusing Sculk Raid code to spread Warped or Crimson across the nether wastes and forests (soulsand valley and basalt delta will get their own special attention another time)
- v2.2 "NotTheEnd" - TBD (idk what to put here yet, but it won't be any End Update stuff)
